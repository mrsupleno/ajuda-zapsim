# API — SuperAdmin (Tenant Management)

> ⚠️ Estas rotas são da API do **SuperAdmin** e usam autenticação diferente (token superadmin).

## Base URL — SuperAdmin

```
https://{seu_backend}/v2/api/superadmin/
```

## Criar API de um Tenant

```
POST /tenantCreateApi
```

## Deletar API de um Tenant

```
POST /tenantDeleteApi
```

## Criar Tenant

```
POST /tenantApiStoreTenant
```

**Body JSON:**

```json
{
  "name": "Empresa Cliente",
  "email": "admin@empresa.com",
  "password": "senha123",
  "plan": "basic"
}
```

**Planos disponíveis:**

| Plano | Descrição |
|-------|-----------|
| `basic` | Plano básico |
| `professional` | Plano profissional |
| `enterprise` | Plano enterprise |

## Buscar Tenant

```
POST /tenantApiShowTenant
```

**Body JSON:**

```json
{
  "tenantId": 10
}
```

## Atualizar Tenant

```
POST /tenantApiUpdateTenant
```

**Body JSON:**

```json
{
  "tenantId": 10,
  "name": "Novo Nome",
  "plan": "professional"
}
```

## Listar Tenants

```
GET /tenantApiListTenants
```

## Criar Sessão para Tenant

```
POST /tenantApiCreateSession
```

## Gerenciamento de Planos

| Recurso | Basic | Professional | Enterprise |
|---------|-------|--------------|------------|
| Canais | 1 | 3 | Ilimitado |
| Usuários | 2 | 10 | Ilimitado |
| Campanhas | ❌ | ✅ | ✅ |
| API | ✅ | ✅ | ✅ |
| Chatbot Nativo | ✅ | ✅ | ✅ |
| Integração IA | ❌ | ✅ | ✅ |

## Autenticação SuperAdmin

O token SuperAdmin é diferente do token de tenant. Obtido no painel SuperAdmin.

```http
Authorization: Bearer {superadmin_token}
```

# API — Templates WABA

> ⚠️ **Apenas para canais WhatsApp Oficial (WABA).** Templates precisam ser aprovados pela Meta.

## Enviar Template com Body

```
POST /v2/api/external/{ApiID}/templateBody
```

**Body JSON (exemplo):**

```json
{
  "number": "5511999999999",
  "templateName": "confirmacao_pedido",
  "language": "pt_BR",
  "components": [
    {
      "type": "body",
      "parameters": [
        { "type": "text", "text": "João Silva" },
        { "type": "text", "text": "Pedido #1234" }
      ]
    }
  ]
}
```

## Componentes Disponíveis

| Tipo | Descrição |
|------|-----------|
| `body` | Corpo do template com variáveis |
| `header` | Cabeçalho (texto ou mídia) |
| `footer` | Rodapé com texto fixo |
| `buttons` | Botões de ação |

# API — Tickets

## Criar Ticket

```
POST /v2/api/external/{ApiID}/createTicket
```

**Body JSON:**

```json
{
  "contactId": 123,
  "queueId": 1,
  "userId": 5,
  "status": "open"
}
```

## Atualizar Fila do Ticket

```
POST /v2/api/external/{ApiID}/updatequeue
```

**Body JSON:**

```json
{
  "ticketId": 456,
  "queueId": 2
}
```

## Atualizar Informações do Ticket

```
POST /v2/api/external/{ApiID}/updateticketinfo
```

**Body JSON (campos possíveis):**

```json
{
  "ticketId": 456,
  "status": "closed",
  "userId": 3,
  "queueId": 1
}
```

**Status disponíveis:** `open`, `pending`, `closed`

## Buscar Informações do Ticket

```
POST /v2/api/external/{ApiID}/showticket
```

**Body JSON:**

```json
{
  "ticketId": 456
}
```

## Buscar Todos os Tickets do Contato

```
POST /v2/api/external/{ApiID}/showallticket
```

**Body JSON:**

```json
{
  "contactId": 123
}
```

## Listar Mensagens de um Ticket

```
POST /v2/api/external/{ApiID}/showAllMessages
```

**Body JSON:**

```json
{
  "ticketId": 456
}
```

## Criar Nota Interna

```
POST /v2/api/external/{ApiID}/createNotes
```

**Body JSON:**

```json
{
  "ticketId": 456,
  "note": "Cliente solicitou retorno até sexta-feira"
}
```

## Listar Notas

```
GET /v2/api/external/{ApiID}/listNotes?ticketId=456
```

## Adicionar Tag ao Ticket

```
POST /v2/api/external/{ApiID}/addTag
```

**Body JSON:**

```json
{
  "ticketId": 456,
  "tagId": 7
}
```

## Enviar Presença (digitando...)

```
POST /v2/api/external/{ApiID}/sendPresence
```

**Body JSON:**

```json
{
  "ticketId": 456,
  "type": "composing"
}
```

| Type | Descrição |
|------|-----------|
| `composing` | Digitando |
| `recording` | Gravando áudio |
| `paused` | Parou |

## Fluxo Típico de Atendimento via API

1. **Receber mensagem** → via webhook ou poll
2. **Criar ticket** → `POST /createTicket`
3. **Atender** → `POST /updateticketinfo` (atribuir userId)
4. **Responder** → `POST /` (enviar mensagem)
5. **Fechar** → `POST /updateticketinfo` com `status: "closed"`

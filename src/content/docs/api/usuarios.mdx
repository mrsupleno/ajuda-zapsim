# API — Usuários

## Criar Usuário

```
POST /v2/api/external/{ApiID}/createUser
```

**Body JSON:**

```json
{
  "name": "Maria Operadora",
  "email": "maria@empresa.com",
  "password": "senha123",
  "profile": "user"
}
```

### Perfis Disponíveis

| Perfil | Descrição |
|--------|-----------|
| `admin` | Administrador — acesso completo |
| `supervisor` | Supervisor — campanhas e relatórios |
| `user` | Usuário/Agente — atendimento |

## Atualizar Usuário

```
POST /v2/api/external/{ApiID}/updateUser
```

**Body JSON:**

```json
{
  "userId": 5,
  "name": "Nome Atualizado",
  "email": "novo@email.com"
}
```

## Listar Usuários

```
GET /v2/api/external/{ApiID}/listUsers?pageNumber=1&searchParam=Maria
```

**Parâmetros de query:**

| Parâmetro | Descrição |
|-----------|-----------|
| `pageNumber` | Número da página |
| `searchParam` | Busca por nome ou email |

## Status do Usuário

```
GET /v2/api/external/{ApiID}/getUserStatus?userId=3
```

**Resposta típica:**

```json
{
  "userId": 3,
  "name": "João Operador",
  "status": "online",
  "totaltickets": 15,
  "pendingtickets": 3
}
```

# API — Usuários

## Criar Usuário

```
POST /v2/api/external/{ApiID}/createUser
```

**Body JSON:**

```json
{
  "name": "Maria Operadora",
  "email": "maria@empresa.com",
  "password": "senha123",
  "profile": "user"
}
```

### Perfis Disponíveis

| Perfil | Descrição |
|--------|-----------|
| `admin` | Administrador — acesso completo |
| `supervisor` | Supervisor — campanhas e relatórios |
| `user` | Usuário/Agente — atendimento |

## Listar Usuários

```
GET /v2/api/external/{ApiID}/listUsers?pageNumber=1&searchParam=Maria
```

## Status do Usuário

```
GET /v2/api/external/{ApiID}/getUserStatus?userId=3
```

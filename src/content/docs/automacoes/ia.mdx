# Integração com Inteligência Artificial

## ChatGPT (OpenAI)

### Habilitar ChatGPT

**Passo 1 — Ativar globalmente:**
1. Configurações → Bots e IA → Chat GPT
2. Ativar "Ativar integração com ChatGPT"
3. Opcional: "Ativar para todos os atendimentos por padrão"

**Passo 2 — Configurar em um Canal:**
1. Canais → Editar canal desejado
2. Preencher na aba ChatGPT:
   - `ChatGPT API Key` — chave da OpenAI
   - `ChatGPT Organization Key` — ID da organização OpenAI

**Passo 3 — Definir comportamento:**

Opção A — **Prompt direto no ZapSim:**
- Campo `Prompt ChatGPT`: instrução de comportamento, personalidade, regras de negócio

Opção B — **Assistant ID da OpenAI:**
- Campo `AssistantId ChatGPT`: ID de um Assistant criado no Playground da OpenAI
- ⚠️ Se usar Assistant ID, o campo Prompt será ignorado

### Configurações Adicionais

| Campo | Descrição |
|-------|-----------|
| `Palavra para desligar ChatGPT` | Ex: "falar com atendente" — desativa bot e chama humano |
| `ChatGPT Model` | Modelo LLM (ex: gpt-4o, gpt-4, gpt-3.5-turbo) |
| `ChatGPT Voice Model` | Modelo de voz para respostas em áudio |
| `Ativar Respostas de Voz` | Bot responde com áudios ao invés de texto |

## Claude (Anthropic)

O mesmo processo do ChatGPT se aplica:

1. Configurações → Bots e IA → Claude
2. Ativar "Ativar integração com Claude"
3. Preencher `Claude API Key`
4. Definir prompt de comportamento

1. Configurações → Bots e IA → Grok

## Ollama (Self-Hosted)

Para modelos Ollama rodando localmente:

1. Configurações → Bots e IA → Ollama
2. Ativar "Ativar integração com Ollama"
3. Preencher URL do servidor Ollama (ex: `http://localhost:11434`)

## LM Studio (Self-Hosted)

1. Configurações → Bots e IA → LM Studio
2. Ativar "Ativar integração com LM Studio"
3. Preencher URL do servidor LM Studio

## Dify

1. Configurações → Bots e IA → Dify
2. Ativar "Ativar integração com Dify"
3. Preencher URL e API Key do Dify

## DialogFlow (Google)

1. Configurações → Bots e IA → DialogFlow
2. Ativar "Ativar integração com DialogFlow"
3. Fazer upload das credenciais Google (JSON)

## Qwen (Alibaba)

1. Configurações → Bots e IA → Qwen
2. Ativar "Ativar integração com Qwen"
3. Preencher `Qwen API Key`

## Boas Práticas para IA

1. **Prompt claro:** Defina comportamento, tom e regras de negócio
2. **Fallback:** Configure palavra para transferir para atendente humano
3. **Limite de tokens:** Evite respostas muito longas
4. **Teste:** Teste diferentes cenários antes de ativar em produção

# Integração N8N

## Habilitar N8N no ZapSim

### Passo 1 — Ativar globalmente

1. Painel Admin → Configurações → Bots e IA → N8N
2. Ativar "Ativar integração com N8N"

### Passo 2 — Configurar em um Canal

1. Editar o canal → Aba N8N
2. Preencher URL do webhook N8N

## Como Funciona

Quando uma mensagem chega no ZapSim, ele envia os dados para o webhook N8N configurado.

## Payload Enviado ao N8N

```json
{
  "event": "message",
  "ticketId": 456,
  "contactId": 123,
  "contactName": "João Silva",
  "contactNumber": "5511999999999",
  "body": "Texto da mensagem"
}
```

## Responder via N8N

```http
POST https://{backend}/v2/api/external/{ApiID}
Authorization: Bearer {token}

{
  "number": "5511999999999",
  "body": "Resposta automática via N8N"
}
```

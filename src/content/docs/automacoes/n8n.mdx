# Integração N8N

## Habilitar N8N no ZapSim

### Passo 1 — Ativar globalmente

1. Painel Admin → Configurações → Bots e IA → N8N
2. Ativar "Ativar integração com N8N"
3. Opcional: "Ativar para todos os atendimentos por padrão"

### Passo 2 — Configurar em um Canal

1. Editar o canal → Aba N8N
2. Preencher:
   - URL do webhook N8N
   - Token de autenticação (se necessário)

## Como Funciona

Quando uma mensagem chega no ZapSim, ele envia os dados para o webhook N8N configurado. O N8N processa e pode responder de volta via API do ZapSim.

## Payload Enviado ao N8N

```json
{
  "event": "message",
  "ticketId": 456,
  "contactId": 123,
  "contactName": "João Silva",
  "contactNumber": "5511999999999",
  "body": "Texto da mensagem",
  "whatsappId": 1,
  "queueId": 2,
  "userId": null
}
```

## Responder via N8N

O N8N pode responder usando a API do ZapSim:

```http
POST https://{backend}/v2/api/external/{ApiID}
Authorization: Bearer {token}
Content-Type: application/json

{
  "number": "5511999999999",
  "body": "Resposta automática via N8N"
}
```

## Casos de Uso

| Caso | Descrição |
|------|-----------|
| **Respostas dinâmicas** | Consultar banco de dados e responder com dados reais |
| **CRM externo** | Sincronizar com Salesforce, HubSpot, etc |
| **E-commerce** | Consultar pedidos, status de entrega |
| **ERP** | Integrar com sistemas de gestão |
| **Agendamentos** | Verificar disponibilidade e agendar |

## Configurações Disponíveis

| Opção | Descrição |
|-------|-----------|
| **Ativar integração com N8N** | Permite uso do N8N no sistema |
| **Ativar para todos os atendimentos por padrão** | N8N ativo em todos os canais automaticamente |

## Boas Práticas

1. **Timeouts:** Configure timeouts apropriados no N8N
2. **Retentativas:** Use a opção de retentativas no webhook do chatbot
3. **Validação:** Valide os dados recebidos antes de processar
4. **Log:** Mantenha logs para debugging

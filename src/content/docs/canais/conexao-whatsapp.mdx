# Conexão WhatsApp

## Tipos de Canal Disponíveis

| Tipo | Descrição | Estabilidade |
|------|-----------|--------------|
| **Baileys (QR Code)** | Conexão via biblioteca Baileys, escanear QR | Boa, depende do smartphone |
| **WebJS** | Conexão via WhatsApp Web JS | Similar ao Baileys |
| **Meow** | Conexão alternativa | Variável |
| **Evolution API** | Integração com Evolution API externa | Alta |
| **Uazapi** | Integração com Uazapi | Alta |
| **WhatsApp Oficial (WABA)** | API Cloud Meta, números certificados | Máxima |

## Como Conectar — Baileys (QR Code)

**Pré-requisitos:**
- Remover conexão ativa no WhatsApp Web antes
- Manter celular ligado, conectado à internet, app WhatsApp aberto
- Desativar descanso de tela no celular
- Recomendado: Chrome + Windows

**Passos:**
1. Painel Admin → Canais → Adicionar Canal
2. Selecionar tipo: `Baileys`
3. Preencher Nome do Canal
4. Salvar → Clicar em "Novo QR Code"
5. No celular: WhatsApp → Configurações → Aparelhos Conectados → Conectar Aparelho
6. Escanear QR Code exibido na tela

**Opção alternativa — Pairing Code:**
- Informar número completo (DDI + DDD + número, sem caracteres especiais)
- O código será enviado diretamente para o celular

## Como Conectar — WhatsApp Oficial (WABA)

1. Necessário ter conta na Meta Business Manager (BM)
2. Preencher: Phone Number ID, WABA ID, Token de acesso Meta
3. Seguir guia específico para configuração WABA

## Comparativo: Baileys vs WABA

| | Baileys | WABA (Oficial) |
|---|---------|----------------|
| Aprovação Meta | Não | Necessário |
| Templates | Não | Sim |
| Botões/Listas interativos | Não | Sim (até 3 botões, 10 listas) |
| Risco de banimento | Médio | Baixo |
| Custo | Grátis | Por mensagem |
| Disparo em massa | ⚠️ Risco | ✅ Seguro |
| Necessita celular | Sim | Não |

## Configurações do Canal (após criar)

| Opção | Descrição |
|-------|-----------|
| **ChatBot** | Vincular fluxo de chatbot padrão |
| **Fila** | Fila padrão de atendimento |
| **Usuário** | Atendente padrão |
| **Avaliação automática** | Envia pesquisa de satisfação ao fechar ticket |
| **Transcrição de áudio** | Converte áudios recebidos para texto |
| **Auto distribuir** | Balanceia tickets entre agentes online |
| **Destruição de mensagem** | Remove mensagens deletadas do banco |
| **Mensagem de aniversário** | Envia parabéns automático na data de nascimento |
| **Desligar IA ao abrir ticket** | Desativa ChatGPT/Typebot/N8N quando humano assume |
| **Espera no processamento** | Evita conflito quando cliente manda várias msgs seguidas |
| **Palavra de fechamento automático** | Palavra-chave que fecha o ticket automaticamente |
| **Mensagem de despedida** | Mensagem padrão ao encerrar atendimento |

## Erros de Conexão Baileys

- ❌ Não manter celular conectado → Reconectar manualmente
- ❌ Abrir WhatsApp Web no navegador ao mesmo tempo → Desconecta a sessão
- ❌ Celular sem bateria/internet → Canal fica offline
- ✅ Solução: Celular sempre carregando, internet estável, app aberto, tela não desligando

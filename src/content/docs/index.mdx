# ZapSim — Documentação Completa

**ZapSim** é uma plataforma SaaS de **atendimento multicanal** focada em WhatsApp. Permite gerenciar múltiplos números, criar chatbots, integrar IAs e automatizar atendimentos.

## Arquitetura

### Perfis de Acesso

| Perfil | Acesso |
|--------|--------|
| **SuperAdmin** | Cria e gerencia tenants (empresas), acesso a todos os painéis |
| **Administrador** | Configura canais, filas, chatbots, campanhas, API do tenant |
| **Supervisor** | Acesso a campanhas, disparo em massa, relatórios |
| **Usuário/Agente** | Atende tickets, acessa conversas |

### Conceitos-Chave

- **Tenant:** Uma empresa/conta no sistema. Cada tenant tem seus próprios canais, usuários, API keys.
- **ApiID:** ID único da API do tenant. Usado em todas as rotas `/v2/api/external/{ApiID}/...`
- **Bearer Token:** Token JWT de autenticação. Obtido via login na API.
- **Canal/Conexão:** Um número de WhatsApp conectado. Cada canal tem um ID.
- **Ticket:** Um atendimento (conversa com um contato). Tem status: `open`, `pending`, `closed`.
- **Fila:** Departamento/setor para organizar tickets (ex: Suporte, Vendas).
- **Tag:** Etiqueta aplicável a contatos ou tickets.

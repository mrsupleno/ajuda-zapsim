## WhatsApp desconecta sozinho

**Causas:**
- Celular sem internet
- Bateria descarregou
- WhatsApp Web aberto em outro navegador

**Solução:** Celular conectado, carregando, app aberto

## Erro 401 Unauthorized (API)

Token inválido ou expirado. Renovar no painel Admin → API
